## Appendix: SQL Queries

### Age Breakdown
```sql
SELECT
  JSON_VALUE(raw_metrics, '$.age') as age_range,
  SUM(impressions) as impressions,
  ROUND(SUM(impressions) / SUM(SUM(impressions)) OVER () * 100, 1) AS pct_of_total,
  SUM(clicks) as clicks,
  ROUND(AVG(SAFE_DIVIDE(clicks, NULLIF(impressions, 0)) * 100), 2) as ctr_pct
FROM `puttery-golf-001.paid_social.fct_ad_insights_daily`
WHERE date BETWEEN '2025-04-01' AND '2025-05-12'
  AND JSON_VALUE(raw_metrics, '$.age') IS NOT NULL
GROUP BY age_range
ORDER BY impressions DESC;
```

### Gender Performance

```sql
  AND JSON_VALUE(raw_metrics, '$.gender') IS NOT NULL
```

### Best Performing Segments
